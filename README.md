# Smart Daily Activity Tracker


## Introduction
In today’s fast-paced life, managing daily activities efficiently has become a challenge. Many students struggle to maintain consistency in their religious obligations, academic studies, and skill development. This project aims to develop a Smart Daily Activity Tracker that helps users monitor and improve their daily routines.

---

##  Features

- 🕌 Track daily Namaz (prayers)
- 📚 Record study hours and subjects
- 💡 Monitor skill development progress
- 📅 Daily activity logging
- 🗄️ Data stored using MySQL database
- 🌐 User-friendly web interface

---

## Technologies Used

- **Frontend:** HTML, CSS, JavaScript  
- **Backend:** PHP  
- **Database:** MySQL  
- **Server:** XAMPP (Localhost)
